Сайт за 5 минут 🚀

Гайд "на пальцах": от регистрации до первой ссылки в сети

1. Регистрация

GitHub — это "облако" для твоего кода. Без него сайт в интернет не выложить.

💡 Почту лучше брать ту, которой пользуешься постоянно.

2. Место для сайта

Проекты на GitHub называются "репозиториями". Это как папка, но в сети.

💡 Ты можешь выбрать Private — сайт все равно будет работать, но код никто не увидит.

3. Загрузка файлов

Теперь нужно перекинуть твой код с компа на GitHub.

💡 Главный файл обязательно должен называться index.html (с маленькой буквы).

4. Запуск хостинга

Делаем так, чтобы сайт открывался по ссылке, а не как файл.

💡 Подожди 2-3 минуты. Ссылка "Your site is live..." появится вверху этой же страницы.

🎉 Молодец!

Ты зарегался, создал репозиторий и выложил свой сайт. Теперь ты официально веб-разработчик. Туда-сюда и ты уже профи! Делись ссылкой с друзьями.
